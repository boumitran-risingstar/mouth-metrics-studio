# **App Name**: Mouth Metrics Studio

## Core Features:

- Phone Authentication: Secure user authentication using phone number verification.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and intelligence.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#009688) to highlight key metrics and calls to action.
- Body and headline font: 'Inter', a sans-serif font, will be used for a clean, modern and readable experience.
- Use professional and clear icons related to speech, analysis, and metrics.
- Subtle animations and transitions to enhance user experience during data loading and display.
